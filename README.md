# Data Analysis Professional Certification

Build job-ready data analysis skills from scratch—no coding or math experience needed.  
Inspired by industry data science roadmaps, this program equips you to clean, analyze, visualize, and communicate data insights using tools like Python, Excel, SQL, and Power BI. Perfect for beginners, career switchers, students, NYSC members, and professionals.

## Program Overview
- **Target Audience**: Beginners with basic computer skills, university students/graduates, career switchers, NYSC members, professionals seeking data skills.
- **Prerequisites**: Basic computer literacy (Windows, files, typing). No programming or math required.
- **Duration**: 4-6 months (self-paced, 8-12 hours/week).
- **What You'll Learn**: Collect/clean data, write SQL queries, build dashboards, perform EDA, create visualizations, and prep for interviews.
- **Tools**: Python, Excel, SQL (MySQL/PostgreSQL/SQLite), Power BI/Tableau intro, GitHub.
- **Certification**: Data Analysis & Insights Professional Certificate upon completing all courses, quizzes (80%+), and capstone.
- **Career Outcomes**: Data Analyst, Junior Data Scientist, Business Analyst, Reporting Analyst, Research Analyst.

Enrollment includes: Video lectures, downloadable resources, quizzes, coding labs, discussion forums, peer reviews, and lifetime access.

## Courses

### Course 1: Maths & Statistics Foundations for Data Analysis
**Duration**: 2 weeks | **Level**: Beginner  
**Goal**: Master statistical thinking for real-world data decisions.  
**What You'll Learn**
- Understand data types and descriptive stats.
- Calculate key metrics and interpret distributions.
- Differentiate correlation from causation.  

**Modules**
1. What is Data? (Categorical vs. Numerical)
2. Descriptive Statistics: Mean, Median, Mode
3. Measures of Spread: Range, Variance, Std Dev
4. Data Distributions & Percentages
5. Correlation vs. Causation & Intro to Probability
6. Reading Charts & Visual Interpretation  

**Hands-On**
- Quiz: 20 questions on stats basics.
- Assignment: Analyze a small dataset (e.g., sales data) and summarize insights in Excel.  

**Tools**: Calculator, Excel.
